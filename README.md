 一、项目简介
本项目基于OCEMOTION口语情感数据集搭建中文二分类情感分析系统，整体技术栈：`jieba分词 + TF-IDF特征提取 + 逻辑回归LogisticRegression`
1. 数据集总量3.5万条口语文本，按照训练集:验证集:测试集 = 8:1:1分层划分
2. 原生数据集存在正负样本不均衡问题（正面样本仅占28%），通过`class_weight='balanced'`类别权重均衡优化，正面样本召回率由43%提升至65%，模型均衡性大幅改善
3. 基准模型测试集整体准确率可达81.55%
4. 配套可视化评估图表、文本预测记录持久化存储、实时交互式预测功能

 四、数据集说明
1. OCEMOTION数据集：口语化中文情感语料，原生多标签分类
   正面标签：`happiness（快乐）、love（喜爱）、surprise（惊喜）` → 统一映射标签1
   其余情绪（悲伤、愤怒、恐惧等）全部归为负面 → 统一映射标签0
2. 样本分布：负面样本占比72%，正面样本仅28%，属于典型类别不平衡数据集
3. 数据预处理流程：
   1. 清洗非中文字符、统一小写
   2. jieba精确模式分词，空格分隔分词结果
   3. 剔除空文本脏数据，保证训练数据有效性

 五、核心模型流程
1. 分层数据集拆分（防止随机划分分布偏移）
   第一步：总数据80%训练集 + 20%临时集
   第二步：临时集对半拆分，各10%作为验证集、测试集
   全程使用`stratify`分层抽样，保证三集合正负样本比例和原数据集一致
2. 特征工程
   仅用训练集拟合TF-IDF词袋词典，验证/测试集只做特征转换，彻底杜绝数据泄露
3. 模型训练与调优
   基础模型：逻辑回归 `LogisticRegression(max_iter=1000)`
   关键优化：`class_weight='balanced'` 自动加权平衡正负样本损失
4. 多维度评估体系
   量化指标：训练/验证/测试集三路准确率、精确率、召回率、F1分数
   可视化指标：准确率对比图、混淆矩阵热力图、样本分布统计图

2. 直接运行主脚本 `Sentiment Analysis.py`
3. 程序自动执行流程：
   读取数据 → 文本清洗分词 → 标签二分类映射 → 三层数据集划分 → TF-IDF特征构建 → 加权逻辑回归训练 → 全套指标计算 → 文本+图片结果保存 → 开启交互式预测终端

七、模型效果指标
1. 基础无权重模型：整体准确率81.55%，正面样本召回率仅43%
2. 加权平衡优化后：
      测试集整体准确率：81.55%
      正面情感召回率提升至65%，正负样本预测均衡性显著提升
3. 输出完整分类报告（精确率Precision / 召回率Recall / F1-Score）、混淆矩阵直观查看错分情况
